# 《世界观设定》（含玩法对应）

> 一句话玩法核心：**尽可能延缓被空白吞噬**  
> 关键词：阿尔兹海默症 / 清醒与发作 / 记忆碎片 / 场景变幻 / 空白侵蚀 / 儿子作为现实锚点  
> 叙事目标：让玩家理解自己在维持一段“尚能被称作自我”的时间，并目睹空白如何逐步覆盖意义。

---

## 1. 世界观总述（两层结构）

### 1.1 现实层
一位患有阿尔兹海默症的老人，由儿子一家照顾。老人处在“间歇性清醒”与“发作”之间的摆荡：  
- 清醒时，他知道自己正在失去心智，并能感知这种失去带来的恐惧。  
- 发作时，他对现实的理解发生扭曲；关怀会被翻译成压力，解释会被翻译成逼迫。

### 1.2 妄想层（游戏主体）
发作时，老人的内在体验呈现为一组**不断变幻的场景**：它们由人生记忆碎片即时拼接、重组、错位而成。  
- 空间关系与时间顺序会漂移，熟悉的物会失去稳定意义。  
- 场景的变化体现意识的失稳与病症波动。  
- 贯穿一切的力量是**空白**：被遗忘所形成的缺口在持续扩张，吞噬记忆与意义。

---

## 2. 核心主题与叙事立场

### 2.1 空白
空白是逐步扩大的缺失：  
- 记忆断裂、意义消散、命名能力坍塌。  
- 恐惧、愧疚、遗憾作为内在压力回流，并被空白不断覆盖、改写。

### 2.2 躲避
躲避是一种求生本能：  
- 每一次成功躲避都换取片刻稳定与更长的维持时间。  
- 生存越久，越能感受到空白从边缘向中心推进的侵蚀过程。

### 2.3 儿子的位置（现实锚点）
儿子是现实世界的锚点：  
- 是老人的名字偶尔能唤回“清醒”的声音来源。  
- 也是照护行为与陪伴关系在妄想层里的稳定信号。  
- 当老人发作时，这份关怀仍会被疾病扭曲地翻译，形成矛盾的体验：被守护与被压迫交织出现。

---

## 3. 玩法—世界观对应（总映射）

> 设计原则：玩家操作的每一个系统，都能被解释为现实照护/病症体验在妄想层中的投射。  
> 即：**现实事件 → 妄想翻译 → 游戏机制反馈**。

### 3.1 核心（Core）
- **世界观含义**：老人仅存的自我感与人格边界（“我是谁”“我在这里”）。  
- **玩法含义**：需要被维持的“清醒残余”；核心状态代表自我完整度。  
- **叙事表达**：核心承受的是“意义被抹去”的侵蚀压力；当核心崩坏，代表这一段维持结束，空白占据主导。

### 3.2 护盾（Shield）
- **世界观含义**：儿子一家的守护与陪伴，是现实锚点在妄想层里的可感知形态。  
- **玩法含义**：围绕核心运转，提供边界与缓冲，帮助老人维持自我稳定。  
- **叙事表达**：护盾的变化体现照护强度、沟通有效性、以及老人当下对关怀的可接受程度。

### 3.3 弹幕（威胁实体）
- **世界观含义**：发作时大脑对外界刺激与内在情绪的“威胁化翻译”。  
  - 过量的信息、突然的声音与光线、无法理解的指令、未被回应的倾诉、压抑的恐惧与愧疚，都可被转译为迫近的压力轨迹。  
- **玩法含义**：需要躲避的压力模式；弹幕的密度、节奏与角度变化体现症状强弱与波动形态。  
- **叙事表达**：被命中意味着空白进一步侵入自我边界，稳定性下降。

### 3.4 升级（三选一能力）
- **世界观含义**：现实事件在妄想层的投射，是“照护变化/病情波动/关系波动”被翻译为规则改变的瞬间。  
- **玩法含义**：玩家通过选择来适应下一段发作走向，以延缓空白吞噬。  
- **示例（设定口径）**：  
  - “病情加重”触发儿子的急迫 → 对应护盾旋转加速（守护更急、更紧）。

### 3.5 场景变幻（妄想层的空间表现）
- **世界观含义**：记忆碎片在当下不断重组，意识以“场景切换/错位/变形”的方式呈现自身失稳。  
- **玩法含义**：场景变换用于表达发作阶段的切换与空白侵蚀的推进，并为弹幕模式与护盾状态提供语境。  
- **叙事表达**：玩家感受到的是“我还能否辨认此刻的世界”，以及“熟悉感如何被剥离”。

### 3.6 生存时间（目标指标）
- **世界观含义**：你延长的是一段“仍能被称作自我”的时间。  
- **玩法含义**：核心目标是维持与延缓：在压力轨迹中保持边界、争取更多时间。  
- **叙事表达**：坚持越久，越清晰地看见空白如何从边缘扩张并覆盖中心。

---
### 3.7 经验系统（XP / 认知残余）
- **世界观含义**：XP 代表老人尚能保有的**认知残余**（cognitive residue）——未被空白完全吞噬的记忆碎片与自我片段。每一次"升级"对应妄想层中一次短暂的认知清晰稳定期，老人能在这段窗口内更清晰地感知自我。
- **玩法含义**：积累 XP 触发升级，升级时获得能力选择。经验球的收集与积累映射"在遗忘过程中偶然保留的认知颗粒"。
- **叙事表达**：经验球应设计为微小的记忆碎片（老照片、常用语、家人的脸）在眼前掠过后被短暂捕捉的画面。暴击格挡（crit_block）代表格挡压力时偶然保留了更多认知残余。
- **经验球到升级的流程**：认知残余积累至阈值 → 触发一次认知稳定窗口 → 老人短暂获得选择"照护方向"的能力（三选一升级）→ 窗口结束后继续下一段维持。


## 4. 叙事表达口径（用于文案/美术/演出统一）

- **维持感**：一切反馈服务于“撑住、稳定、延缓”的情绪。  
- **空白感**：侵蚀被表达为褪色、缺口、静音、断句、失配与不可名状的留白。  
- **锚点感**：儿子的声音与照护信号具有一致性与辨识度，在��乱中提供短暂方向。

---

## 5. 世界观一致性规则（新增内容时使用）

新增任何玩法/系统/事件时，至少回答三句：
1. **现实层对应什么？**（照护行为/病情变化/关系波动/外界刺激）  
2. **在妄想层被翻译成什么？**（弹幕模式/护盾行为/场景异化/空白扩张方式）  
3. **它如何服务“尽可能延缓被空白吞噬”？**（提供稳定窗口、改变压力节奏、延长维持时间）
